# Pokémon TCG Store

A professional store for Pokémon Trading Card Game (TCG) with the following features:

- Complete database of 100+ cards
- Advanced shopping cart
- User authentication
- Product filters
- Reviews system
- Admin panel
- Payment integration
- Responsive design

## Features
- Manage and view cards
- Users can create accounts and log in
- Cart functionality for shopping
- Admin tools for managing products and users
